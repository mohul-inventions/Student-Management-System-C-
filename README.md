# C++ Student Management System

A console-based Student Management System built in C++ that efficiently manages student records using file handling and menu-driven operations. 

This project was developed as Task 1 for the C++ Developer Internship.

## 🚀 Features Included
* **Add New Students:** Capture a student's Roll Number, Name, and Course.
* **Display Records:** View a formatted list of all registered students.
* **Update Information:** Modify the name and course of an existing student using their Roll Number.
* **Delete Records:** Remove a student from the system.
* **Persistent Storage:** All data is automatically saved to and loaded from a local `students.txt` file, ensuring no data is lost when the program closes.
